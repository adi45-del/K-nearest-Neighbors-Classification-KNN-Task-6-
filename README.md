# K-nearest-Neighbors-Classification-KNN-Task-6-
AI/ML Internship Task 6: Implemented K-Nearest Neighbors (KNN) classification on the Iris dataset with feature normalization, model evaluation, and decision boundary visualization.

## Objective
The objective of this task is to understand and implement the **K-Nearest Neighbors (KNN)** algorithm for classification problems.

## Dataset Used
- **Iris Dataset**
- The dataset contains measurements of iris flowers and is used to classify them into three species:
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Tasks Performed
- Imported and preprocessed the Iris dataset.
- Removed unnecessary columns.
- Split the dataset into training and testing sets.
- Normalized the features using StandardScaler.
- Trained a K-Nearest Neighbors (KNN) classifier.
- Tested different values of **K** to compare model performance.
- Evaluated the model using Accuracy Score and Confusion Matrix.
- Visualized the Decision Boundary.

## Learning Outcome
Through this task, I learned:
- K-Nearest Neighbors (KNN) Classification
- Data Normalization
- Train-Test Split
- Model Evaluation
- Confusion Matrix
- Decision Boundary Visualization
- Choosing the best value of K
